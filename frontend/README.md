# Exercise Form Frontend

Flutter frontend scaffold for the exercise form detection project.

## Main folders

- `lib/app`: app bootstrap and root widget
- `lib/core`: shared constants, routes, theme, helpers
- `lib/features`: feature-wise screens
- `lib/shared`: reusable widgets
- `assets`: images, icons, animations
- `test`: widget tests

## How to run

```
flutter clean
flutter pub get
flutter run
```
